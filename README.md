#My Fitness Jorney

A personal fitness webpage built with HTML and CSS, created as part of a web development learning project.

##Description

This is a single-page website about my personal fitness journey. It introduces who I am, shares my story of getting into fitness, and aims to inspire others to start their own movement journey. The page includes images, a navigation list, and a link to an external fitness resource.

##Purpose

###The goal of this project is to:

-Practise building a real webpage using HTML and CSS
-Learn how to structure a page using semantic HTML tags
-Apply CSS styling including colours, fonts, borders, and layout
-Publish a live website using GitHub Pages
